# Air Quality Data Platform

  ## Overview
  Scalable big data platform for air quality analysis, built on AWS cloud infrastructure.

  ## Architecture
  - **Cloud**: AWS (EC2, S3, EMR)
  - **Ingestion**: Apache NiFi
  - **Processing**: Apache Spark
  - **Storage**: Parquet on S3
  - **Visualization**: Elasticsearch + Kibana

  ## Data Sources
  - OpenAQ (global sensor data)
  - Hava Ä°zleme (Turkish national stations)
  - OpenAlex (287K+ academic papers, ~870GB)
